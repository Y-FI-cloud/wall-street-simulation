# 🗽 Wall Street RPG

Welcome to **Wall Street RPG**, a web-based stock market and cryptocurrency trading simulator built with Python and Streamlit! 

Experience the thrill of trading in a gamified environment where you can buy, short, and manage a portfolio using **real-time market data** powered by Yahoo Finance (`yfinance`). Compete against billionaire bots, earn XP, unlock achievements, and climb the leaderboard!

## ✨ Features

- **Real-Time Trading:** Trade US Stocks, ETFs, and Cryptocurrencies using live market data.
- **Long & Short Positions:** Buy assets if you think they will go up, or short them if you think they will crash.
- **RPG Elements:** Earn XP for every trade, level up, and unlock special achievements.
- **Live Portfolio Management:** Track your profits and losses (P/L) in real-time and close positions partially or fully.
- **Leaderboard & Bots:** Compete against simulated AI bots (like Warren B. and Elon M.) whose wealth fluctuates with the market.
- **Educational:** Includes an integrated encyclopedia for popular ETFs (VOO, QQQ, GLD, etc.) to help you learn while playing.

## 🛠️ Installation & Setup

To run this game locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Y-FI-cloud/wall-street-simulation.git](https://github.com/Y-FI-cloud/wall-street-simulation.git)
   cd wall-street-simulation
   
2. **Install the required libraries:**
Make sure you have Python installed, then run:
```bash
pip install -r requirements.txt
```
3. **Run the Streamlit app:**

 ```bash
streamlit run app.py
```
